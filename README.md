**FTWM 폴더**

설명: FTWM(Fractal Tree Writing Method) 관련 자료와 프롬프트 템플릿을 모아둔 폴더입니다.

파일 구조:
- FTWM-definition-v4.1.md : 원문 요약 및 원문 링크
- prompt_template.md : 프롬프트 템플릿(프롬프트 엔지니어링용)
- examples/prompt_example.txt : 템플릿 예시 입력

Sources (원문/참조)
- sources/FTWM-definition-v4.1-full.md : FTWM 정의(원문 전체) — 원문 Raw 링크 포함
- sources/FTWM-v4.1-Prompt.md : FTWM v4.1 Prompt (원문 링크 및 요약)
- sources/FTWM.Character_v2.1.md : NovelPlotting Character 어댑터(요약 + 원문 링크)
- sources/FTWM.NP-v2.1.md : NovelPlotting PlotElements(요약 + 원문 링크)

사용 방법:
1. `prompt_template.md`를 열어 템플릿 필드를 채웁니다.
2. 내용을 복사하여 AI(또는 로컬 스크립트)에 입력하면 FTWM 설계안이 생성됩니다.
3. 생성된 설계는 `Drilling`(Depth)와 `Diagnostics`를 통해 보강하세요.

원문 참조: FTWM-definition-v4.1.md (원문 Raw 링크 포함)

사용 예시:
1. `sources` 폴더의 각 파일에서 원문 Raw 링크를 클릭하여 전체 스펙을 확인하세요.
2. `prompt_template.md`의 필드를 채운 뒤 AI에게 입력하면 FTWM 워크플로우에 따른 설계안을 얻을 수 있습니다.
