**FTWM 폴더**

설명: FTWM(Fractal Tree Writing Method) 관련 자료와 프롬프트 템플릿을 모아둔 폴더입니다.

파일 구조:
- `sources/FTWM-definition-v4.1.md` : FTWM 정의 요약 및 원문 Raw 링크
- `sources/prompt_template.md` : 프롬프트 템플릿(프롬프트 엔지니어링용)
- `examples/prompt_example.txt` : 템플릿 예시 입력
- `sources/FTWM-v4.1-Prompt.md` : FTWM v4.1 Prompt (원문 링크 및 요약)
- `sources/FTWM.Character_v2.1.md` : NovelPlotting 캐릭터 모듈 요약
- `sources/FTWM.NP-v2.1.md` : NovelPlotting 플롯 요소 리팩토링 요약

사용 방법:
1. `sources/prompt_template.md`를 열어 템플릿 필드를 채웁니다.
2. 내용을 복사하여 AI(또는 로컬 스크립트)에 붙여넣어 FTWM 설계안을 생성하세요.
3. 생성된 설계는 `Drilling`(Depth)와 `Diagnostics` 단계로 보강하세요.

원문 참조: `sources/FTWM-definition-v4.1.md` (원문 Raw 링크 포함)

사용 예시:
1. `sources` 폴더의 각 파일에서 Raw 링크를 열어 전체 스펙을 확인하세요.
2. `sources/prompt_template.md`의 필드를 채운 뒤 AI에 입력하면 FTWM 워크플로우 설계가 출력됩니다.
